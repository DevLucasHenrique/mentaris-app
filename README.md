## 📌 About

**Mentaris** is a fictional mental health clinic built as a portfolio project. The platform features a modern institutional website with online appointment scheduling, patient testimonials, and information about services and treatments.

> ⚠️ This is a fictional project developed for portfolio purposes only.

---

## 🚀 Features

- ✅ Institutional landing page with hero, services and testimonials
- ✅ Online appointment scheduling
- ✅ Virtual store and mobile app sections
- ✅ REST API with full CRUD
- ✅ JWT Authentication
- ✅ Clean Architecture on the backend
- ✅ Containerized with Docker
